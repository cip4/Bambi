This is a short step by step howto build instruction for bambi.

Prerequisites

You will need 
Bambi: http://cip4srv2.igd.fhg.de/svnroot/trunk/Bambi
JDFLib-J: http://cip4srv2.igd.fhg.de/svnroot/trunk/JDFLibJ
Both directory trees should reside in a parallel parent directory.
Tomcat (5.5 or 6.0) - If you use 5.5 make sure taht you complie with Java 1.5 compatibility enabled!


Build JDFLib: run the ant script with the "spam" option enabled
Build Bambi core: run http://cip4srv2.igd.fhg.de/svnroot/trunk/Bambi/build-core.xml
Build the simworker with the install or redeploy option: http://cip4srv2.igd.fhg.de/svnroot/trunk/Bambi/build-simworker.xml
if the option does not work, manually copy Simworker.war to (ApacheHome)/webapps

go!
